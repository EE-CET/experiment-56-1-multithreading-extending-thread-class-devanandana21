# Experiment 56: Multithreading (Extending Thread Class)

## Problem Statement

Create a class `MyThread` that extends `Thread`.
The thread should print "Thread is running".
Write a main method to create and start the thread.

## Input Format

* No input.

## Output Format

* `Thread is running`

### Example 1

**Input:**

```text
No input
```

**Output:**

```text
Thread is running
```
